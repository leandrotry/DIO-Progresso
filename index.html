Módulo 0

Tudo iniciou-se com o Módulo 0, onde houveram alguns conteúdos de autoajuda, as boas vindas ao programa, e também onde foram explicadas algumas métricas do Bootcamp.

Módulo I

Curso de Introdução à Programação e Pensamento Computacional

Já no Módulo I, a professora Juliana Mascarenhas, deu algumas aulas introdutórias ao programa com o curso de pensamento computacional e programação de computadores.

Nesse curso, ela explana como funcionam os computadores na sua parte lógica,apresenta e da exemplos do que é penamento computacional e dá alguns exemplos de pseudocódigo em portugol mostrar o que ela explica no curso.

Curso de Introdução ai Git e GitHub

Após o primeiro curso de pensamento computacional, já fui enviado diretamente para o curso de introdução ao Git e GitHub, onde o professor da vez é o Otávio Reis, que ensina passo-a-passo o que é versionamento de código, quais as necessidades de se versinar código, quem criou e mais.

Alías, enquanto escrevo essse artigo, estou versionando tudo também, para manter praticando tudo que é ensiando. Você pode acompanhar tudo através deste link.

Ao término do curo, já recebemos o desafio de criar o primeiro repositório do GitHub.

Módulo II

Introdução a criação de websites com HTML5 e CSS3

O Módulo II se iniciou com o o curso de HTML5 e CSS3 com o professor Lucas Vilaboim mostrando os primeiros passos para se desenvolver páginas web.

Utilizando as linguágens HTML e CSS, o curso mostra como podemos criar e estilizar páginas web de forma semântica, que faça sentindo tanto para o navegador tanto para quanto quem escreve o código.
